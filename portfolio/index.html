Projeto de portfólio

Sistema de Cadastro de Funcionários

Aplicação web completa para gestão de funcionários e lançamentos financeiros, com geração de relatórios em PDF. Desenvolvida com Spring Boot e PostgreSQL.

A demo roda em plano gratuito e pode demorar até 60 segundos na primeira carga após inatividade.
Ver demo ao vivo Código no GitHub

Funcionalidades

👤

CRUD de funcionários

Cadastro, edição, busca e exclusão com validação de dados.

💰

Lançamentos

Controle de lançamentos vinculados a cada funcionário.

📄

Relatório PDF

Exportação de lançamentos em PDF com iText.

Stack tecnológica

Java 17 Spring Boot 3.5 Spring Data JPA Thymeleaf PostgreSQL Bootstrap 5 iText PDF Docker

Arquitetura de deploy

Backend containerizado no Render (free tier) · Banco Neon PostgreSQL (free tier) · Esta landing page na Vercel.
